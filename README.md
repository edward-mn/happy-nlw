## üì¶ How to [download](https://github.com/edward-mn/happy-nlw/archive/master.zip) and run
```bash
# Clone's reposity
$ git clone git@github.com:edward-mn/happy-nlw.git

# Acess the folder project into your vscode
$ cd happy-nlw

# Install the dependencies
```
